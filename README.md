# POWER-ML-Multi-Model-Machine-Learning-Forecasting-for-NASA-POWER-Data
本项目基于NASA POWER气象数据，构建了完整的数据仓库与ETL流程，运用随机森林、LightGBM和PyTorch三种机器学习算法对地表辐射量进行预测，并修复了实际值与预测值对比图表的显示问题。同时集成了天气模式聚类分析和可视化输出，为气象数据分析与决策支持提供了技术方案。
